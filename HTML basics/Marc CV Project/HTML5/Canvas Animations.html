Working with Canvas The Canvas element can be transformed. As an example, a text is written on the canvas at the coordinates (10, 30). The translate(x,y) method is used to move the Canvas. x indicates how far to move the grid horizontally, and y indicates how far to move the grid vertically. In this example, the canvas is moved 100px to the right, and 150px down. *** Canvas has several methods for drawing paths, boxes, circles, text, and adding images. *** The rotate() Method The rotate() method is used to rotate the HTML5 Canvas. The value must be in radians, not degrees.
